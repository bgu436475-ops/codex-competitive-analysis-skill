## Claim Register

Include exactly one material claim per row and retain every required column below. Use `subject + exactly one audit unit + scoped value/date + source` for each FACT row. Give each strength, weakness, comparison, risk, opportunity hypothesis, and customer need its own INFERENCE row with separate FACT premises. Write each Claim field as the complete proposition reused in the report. For strategic inferences, list two independent exact source links or label confidence lower and explain the gap. Put source IDs in Notes as optional cross-references rather than substituting them for grade, exact link, access date, confidence, or notes.
